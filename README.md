## About ARTCHA

ARTCHA stand for Augmented Reality CAPTCHA is a web CAPTCHA service that using Augmented Reality Technology using 3D object. This project is an example of implementation of ARTCHA. This app build using:

- Laravel v.8.x
- AR.JS
- A-FRAME
- Nexmo
